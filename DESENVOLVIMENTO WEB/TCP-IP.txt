# TCP/IP, PORTAS ,ROTEADORES, SWITCHES, MODEMS

## TCP/IP

- Protocolos de comunição entre computadores em rede
  - Transmission Control Protocol - Protocolo de Controle de Transmissão
  - Internet Protocol - Protocolo de Internet 
- Modelo de camadas
- 4 camadas para começa a transmissão 
  1. Física (ex: placa de rede)
  2. Rede (ex: IP)
  3. Transporte (ex: TCP, UDP)
  4. Aplicação (e: FTP SMTP, HTTP)

## TCP x UDP

- UDP

  - Rápido
  - Não confiável
  - Carro do ovo = comparativo
  - Livestream

- TCP

  - Voltado á conexão
  - Handshake
  - Integridade, ordem dos dados
  - Aplicativo de mensagens de texto = referencia

  Ambos os protocolos já estão configurados na maior parte dos computadores/model/servidores.

  ## PORTA

  Portas (do inglês : ports)

  ​

- As portas por onde os dados sairão ee chegaram

- 20: FTP = APENAS ENVIO DE ARQUIVOS

- 22: SSH = CONEÃO SEGURA 

- 25: SMTP

- 53: DNS

- 80: HTTP = CONEXÃO SIMPLES

- 443: HTTPS

## MODEM
